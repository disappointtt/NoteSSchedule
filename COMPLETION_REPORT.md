## 📱 NoteSSchedule - Реализация готова!

Я создал полнофункциональную реализацию двух экранов приложения на Flutter с современным UI и логикой.

---

### ✅ Что было реализовано:

#### **1. Экран "Задачи" (Tasks Screen)**
- 🎨 Приветствие с текущей датой ("Доброе утро, Диас!")
- 📊 Три цветные статистические пилюли:
  - **7 Задач** (фиолетовая)
  - **2 Встречи** (зелёная)
  - **3 Напоминания** (оранжевая)
  - Click на пилюлю выводит сообщение
- 📅 Полоса выбора дат (Пн-Вс) с выделением текущего дня
  - Переключение между днями обновляет список задач
- 💼 Список моков задач с деталями:
  - Время (09:00, 11:00, 13:00)
  - Название задачи
  - Категория (иконка + цвет)
- 📝 BottomSheet с деталями задачи
  - Кнопка "Отметить выполненной" перемещает в историю
- ➕ Две плавающие кнопки для добавления:
  - "Новая задача" (фиолетовая)
  - "Напоминание" (оранжевая)

#### **2. Экран "Настройки" (Settings Screen)**
- **Основное:**
  - Переключатель уведомлений → сохраняется в Shared Preferences
  - Дата и время (placeholder)
  - Категории задач (placeholder)

- **Оформление:**
  - Выбор темы: Светлая/Тёмная (с немедленным переключением)
  - Три кнопки размера шрифта: A (S), A (M), A (L)
  - Размер сохраняется и влияет на весь текст в приложении

- **Данные:**
  - Информация о локальном хранилище (37.2 МБ)
  - Резервное копирование (placeholder)

- **О приложении:**
  - Поддержка (контакты: email, telegram)
  - О NoteSSchedule (версия и описание)

#### **3. Нижняя навигация**
- 3 таба: Задачи | Календарь | Настройки
- Переключение без потери состояния (IndexedStack)
- Иконки: галочка, календарь, шестеренка

### 🔧 State Management (Provider)

**AppSettings (ChangeNotifier):**
```dart
- notificationsEnabled: bool
- themeMode: light/dark
- fontScale: 0.9 / 1.0 / 1.15
- Сохраняются и восстанавливаются из Shared Preferences
```

**TasksState (ChangeNotifier):**
```dart
- selectedDate: DateTime
- tasksByDate: Map<DateTime, List<Task>>
- reminders: List<Reminder>
- completedHistory: List<Task>
- tasksCount, meetingsCount, remindersCount
```
