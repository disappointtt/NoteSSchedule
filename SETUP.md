# Инструкция по запуску NoteSSchedule

## Требования
- Flutter SDK (version 3.0.0+)
- Dart SDK (встроен в Flutter)
- Android Studio / Xcode (для эмулятора или устройства)

## Установка и запуск

### 1. Установите зависимости
```bash
flutter pub get
```

### 2. Запустите приложение

#### На Android эмуляторе:
```bash
flutter run
```

#### На iOS эмуляторе (macOS only):
```bash
flutter run -d "iPhone 13"
```

#### На физическом устройстве (подключено по USB):
```bash
flutter run
```

#### В режиме разработки с горячей перезагрузкой:
```bash
flutter run -v
```

### 3. Команды разработки

**Анализ кода:**
```bash
flutter analyze
```

**Форматирование кода:**
```bash
flutter pub run dart_style:format .
```

**Очистка проекта:**
```bash
flutter clean
```

**Сборка Release версии (Android):**
```bash
flutter build apk
```

**Сборка Release версии (iOS):**
```bash
flutter build ios
```

## State Management

Приложение использует **Provider** для управления состоянием:

- **AppSettings**: хранит настройки (уведомления, тема, размер шрифта) и загружает их из Shared Preferences при запуске
- **TasksState**: хранит задачи по датам, текущую выбранную дату, напоминания и историю выполненных задач

## Сохранение данных

- Все настройки сохраняются локально с помощью `shared_preferences`
- Рекомендуется добавить SQLite или Hive для сохранения задач (текущая реализация использует памость)
