## 🎯 Интерактивные элементы

### Что кликается и работает:

1. **Пилюли статистики** → Показывают SnackBar
   ```
   7 Задач → "Открыть: Задачи"
   2 Встречи → "Открыть: Встречи"
   3 Напоминания → "Открыть: Напоминания"
   ```

2. **Полоса дат** → Переключают задачи на день
   - Нажмите на `Вт` → список задач обновится на вторник

3. **Задача в списке** → BottomSheet с деталями
   ```
   Нажмите: "09:00 | Совещание с командой"
   ↓
   Детали: название, время, категория, кнопка "Отметить выполненной"
   ```

4. **"Отметить выполненной"** → Задача уходит в историю
   - Счётчик "7 Задач" уменьшится на 1

5. **"Новая задача"** → Диалог добавления
   ```
   Название: [_______________]
   Время: [__: __]
   [Добавить]
   ```

6. **"Напоминание"** → Диалог добавления напоминания
   ```
   Название: [_______________]
   [Добавить]
   ```

7. **Переключатель "Уведомления"** → Сохраняется
   - Выключите → восстановите приложение → остаётся выключено ✓

8. **"Тема"** → Светлая/Тёмная (переключается мгновенно)
   ```
   [Светлая] [Тёмная] ← Нажмите на одну
   Весь интерфейс перекрасится!
   ```

9. **"Размер шрифта"** → S / M / L
   ```
   [A] [A] [A] ← Третья выбрана (L)
   Весь текст становится больше!
   ```

10. **"Поддержка"** → Диалог с контактами
    ```
    📧 Email: support@notesschedule.com
    💬 Telegram: @NoteSSchedule
    ```

11. **"О NoteSSchedule"** → Диалог с версией
    ```
    NoteSSchedule v1.0.0
    Умный планировщик с напоминаниями...
    ```

## 💾 Сохранение данных

### Что сохраняется автоматически:

✅ **Shared Preferences:**
- Уведомления включены/выключены
- Выбранная тема (светлая/темная)
- Размер шрифта (S/M/L)

✅ **Память (текущая сессия):**
- Список задач по датам
- История выполненных задач
- Список напоминаний

_Примечание: Для продакшена добавьте SQLite/Hive для сохранения задач на диск_

## 🐛 Если что-то не работает

1. **"Flutter not found"**
   ```bash
   # Убедитесь что Flutter в PATH
   flutter doctor
   ```

2. **"Packages not found"**
   ```bash
   flutter pub get
   flutter clean && flutter pub get
   ```

3. **"App won't launch"**
   ```bash
   flutter clean
   flutter pub get
   flutter run
   ```

4. **"Build errors"**
   ```bash
   flutter analyze    # Найти ошибки
   flutter pub get    # Переустановить зависимости
   ```
