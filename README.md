## Описание приложения

### Назначение
NoteSSchedule — это мобильное приложение на Flutter, предназначенное для управления личным расписанием, заметками и задачами с интеллектуальной системой напоминаний. Приложение позволяет пользователям планировать свой день, отслеживать важные события и получать своевременные уведомления о начале каждого запланированного действия. Это универсальный инструмент для повышения продуктивности и организации времени, который помогает не пропустить важные дела и события.

### Проблема, которую решает
В современном мире люди часто забывают о запланированных делах, встречах и важных событиях из-за занятости и отвлечений. Существующие приложения-календари часто имеют сложный интерфейс, недостаточное количество функций напоминания или требуют множества действий для добавления события. NoteSSchedule решает эту проблему, предоставляя простой и интуитивный способ организации расписания с автоматическими умными напоминаниями, которые срабатывают точно перед началом каждого события.

### Целевая аудитория
- **Студенты** — для управления расписанием занятий, сдачи работ и экзаменов
- **Работающие люди** — для планирования рабочего дня и встреч
- **Активные люди** — для отслеживания личных задач, хобби и тренировок
- **Люди с плотным графиком** — которые нуждаются в надежной системе напоминаний
- **Возрастная группа:** 16+ лет

---

## Основные функции

- ✅ Создание заметок и задач
- 📅 Умный календарь с расписанием
- 🔔 Автоматические напоминания перед началом события
- 🏷️ Категоризация задач (работа, учеба, личное и т.д.)
- 🎨 Настройка внешнего вида и тем оформления
- 💾 Локальное хранилище данных
- 📊 Просмотр истории выполненных задач
